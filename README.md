# Commands

1. Run Server CMD
```bash
python manage.py runserver
```

2. Create a superuser account
```bash
python manage.py createsuperuser
```

3. Migrate
```bash
python manage.py makemigrations
python manage.py migrate
```

## Git Commands

1. Pull changes
```bash
git pull
```

2. Push changes
```bash
git push
```
